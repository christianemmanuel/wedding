Wedding Guest Attire

Theme

Ang tema natin ay rustic, at ang dress code ay semi-formal. Kung wala po, kahit semi na lang po, charot! Pakiusap po, sumunod sa dress code. Ito po ang color palette na dapat sundan:

Pangbabae

Long dress, sun dress, trousers, jumpsuit—huwag lang po mag-gown, hindi po kayo ang bride. Magdala rin po ng coat/scarf/jacket dahil maaaring ginawin po kayo, hindi po namin sagot kung sakaling lamigin ang inyong mga likod.

Panglalaki

Trousers, long sleeves, slacks, polo shirt, coat—huwag lang po magtsinelas, baka kayo po ay mahampas.
